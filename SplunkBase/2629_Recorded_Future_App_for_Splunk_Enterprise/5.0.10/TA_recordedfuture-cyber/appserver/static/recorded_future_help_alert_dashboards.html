Alert Dashboards

The Alert Dashboard displays current alerts. Alerts are retrieved from the Recorded Future API when the Alert dashboard is accessed or refreshed.

By default the sum of all configured inputs for alerts are shown but specific inputs can be selected using the drop down menu Select an Alert config.

The dashboard contains three fields:

  • Number of Alerts shows how many alerts that are currently active.
  • Counts by Rule displays how many times an alert rule has been triggered.
  • Detailed Alert Information with further information of the alerts.

Click on an alert in Detailed Alert Information to open a new window showing the alert in Recorded Future Portal.
