Explanation of the data in the table

Display name
The display (friendly) name of the component causing delays.
Name
The internal name of the component causing delays.
Version(s
The version number of the component causing delays.
Total startup (s)
Total time it took the component to start in all of the events recorded and analyzed here.
Avg. startup (s)
Average time it took the component to start in all of the events recorded and analyzed here.
Total degradation (s)
Total time it took the component longer than usual to start in all of the events recorded and analyzed here.
Avg. degradation (s)
Average time it took the component longer than usual to start in all of the events recorded and analyzed here.
#Events
Number of start events recorded and analyzed for this component at this specific version.
