Explanation of the data in the table

IP address
IP address found on the Citrix ADC appliance
Netmask
Netmask of the IP address found on the Citrix ADC appliance
Type
Type of the IP address found on the Citrix ADC appliance
Bound to vServer
Indicates if the IP address is bound to a virtual server
Mgmt. access
Indicates if management access is configured for the IP address
