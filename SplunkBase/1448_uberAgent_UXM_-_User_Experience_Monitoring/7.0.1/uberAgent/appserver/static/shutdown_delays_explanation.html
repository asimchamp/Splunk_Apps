Explanation of the data in the table

Display name
The display (friendly) name of the component causing delays.
Name
The internal name of the component causing delays.
Version(s)
The version number of the component causing delays.
Total shutdown (s)
Total time it took the component to shut down in all of the events recorded and analyzed here.
Avg. shutdown (s)
Average time it took the component to shut down in all of the events recorded and analyzed here.
Total degradation (s)
Total time it took the component longer than usual to shut down in all of the events recorded and analyzed here.
Avg. degradation (s)
Average time it took the component longer than usual to shut down in all of the events recorded and analyzed here.
#Events
Number of shut down events recorded and analyzed for this component at this specific version.
