Erklärung der Daten in der Tabelle

App name(s)
Name der Anwendung, zu der der gestartete Prozess gehört.
Process name(s)
Name des Prozesses.
Avg. startup duration (s)
Durchschnittliche Dauer eines Starts in Sekunden.
Avg. startup IO count
Durchschnittliche Anzahl FestplattenIOs pro Start.
#Startups
Anzahl der Startvorgänge im ausgewählten Zeitraum.
#Hosts
Anzahl an Computern, auf denen der Prozess gestartet wurde.
