Explanation of the data in the table

App name(s)
Name of the application the started process belongs to.
Process name(s)
Name of the started process.
Avg. startup duration (s)
Average duration of a start in seconds.
Avg. startup IO count
Average disk IOs per start.
#Startups
Number of starts in the selected time range.
#Hosts
Number of computers the processes were started on.
