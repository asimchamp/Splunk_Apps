Erklärung der Daten in der Tabelle

URL host
Der Hostnamen-Teil der URL, z.B. für die URL "http://helgeklein.com/" wäre dies "helgeklein.com
Total CPU time (s)
Gesamte, von der Site verbrauchte CPU-Zeit.
Avg. CPU (%)
Durchschnittliche, von der Site verbrauchte CPU-Zeit.
Avg. IOPS
Durchschnittlich von der Site erzeugte IOPS.
Total IO (count)
Insgesamt von der Site erzeugte (Lese- und Schreib-) IOPS.
Avg. IO (MB/s)
Durchschnittlich von der Site erzeugtes IO-Volumen in MB/s.
Total IO (MB)
Gesamtmenge des von der Site erzeugten IO-Volumens in MB.
Avg. IO latency (ms)
Durchschnittliche Dauer einer IO-Operation in Millisekunden. Ist leer falls keine IO-Operationen während des gewählten Zeitraums stattfanden.
Avg. RAM (MB)
Durchschnittliche Größe des Working Set (Speicherverbrauchs) der Site.
Avg. network (KB/s)
Durchschnittliche Netzwerk-Übertragungsrate per TCP und UDP im gewählten Zeitraum (IPv4 und IPv6).
