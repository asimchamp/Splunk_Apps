Erklärung der Daten in der Tabelle

User
Name des angemeldeten Benutzers.
Machine
Name des Computers, auf dem die Sitzung lief.
ID
RDS-Sitzungskennung.
Logon time
Anmeldezeitpunkt.
Protocol
Häufigste Art der Verbindung zur Sitzung: console, RDP oder PCoIP.
State
Letzter aufgezeichneter Zustand. Typischerweise active oder disconnected.
Client name(s)
Namen der Clients, von denen aus eine Verbindung zur Sitzung hergestellt wurde.
Client IP(s)
IP-Adressen der Clients, von denen aus eine Verbindung zur Sitzung hergestellt wurde.
Client platform(s)
Plattformen der Clients, von denen aus eine Verbindung zur Sitzung hergestellt wurde.
Client timezone(s)
Zeitzonen der View-Clients, von denen aus eine Verbindung zur Sitzung hergestellt wurde.
Display specs
Anzeigemodus (horizontal x vertical x color depth) der Sitzung auf den Clients.
Pub. name
Name der veröffentlichten Anwendung, die verwendet wurde, um die Sitzung zu starten.
Session type
Gibt an, ob es sich um einen Desktop oder eine Anwendung handelt.
Broker
Name des Sitzungsbrokers.
