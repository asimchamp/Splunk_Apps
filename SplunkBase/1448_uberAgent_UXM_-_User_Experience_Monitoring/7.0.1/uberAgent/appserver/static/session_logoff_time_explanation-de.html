Erklärung der Daten in der Tabelle

User
Name des angemeldeten Benutzers.
Host
Name des Computers, auf dem die Sitzung lief.
Logoff time
Abmeldezeitpunkt
Total duration
Gesamte Abmeldezeit in Sekunden
User profile
Zeit für das Entladen des Benutzerprofils.
GP logoff script
Ausführungszeit des per Gruppenrichtlinien zugewiesenen Abmeldeskripts. Kann leer sein, wenn kein solches Skript existiert.
