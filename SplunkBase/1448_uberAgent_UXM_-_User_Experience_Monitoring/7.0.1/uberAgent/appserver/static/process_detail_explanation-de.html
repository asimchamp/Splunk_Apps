Erklärung der Daten in der Tabelle

Process name
Name des Prozesses.
App name(s)
Name der Anwendung, zu der der Prozess gehört.
Total CPU time (s)
Gesamte, vom Prozess verbrauchte CPU-Zeit.
Avg. CPU (%)
Durchschnittliche vom Prozess generierte CPU-Last.
Avg. IOPS
Durchschnittlich vom Prozess erzeugte IOPS.
Total IO (count)
Insgesamt vom Prozess erzeugte (Lese- und Schreib-) IOPS.
Avg. IO (MB/s)
Durchschnittliches vom Prozess erzeugtes IO-Volumen in MB/s.
Total IO (MB)
Gesamtmenge des vom Prozess erzeugten IO-Volumens in MB.
Avg. IO latency (ms)
Durchschnittliche Dauer einer IO-Operation in Millisekunden. Ist leer falls keine IO-Operationen während des gewählten Zeitraums stattfanden.
Avg. RAM (MB)
Durchschnittliche Größe des Working Set (Speichernutzung) des Prozesses.
Avg. network thruput (KB/s)
Durchschnittliche Netzwerk-Übertragungsrate per TCP und UDP im gewählten Zeitraum (IPv4 und IPv6).
Avg. network latency (ms)
Durchschnittliche Latenz von TCP-Verbindungsaufbau und TCP-Sendeoperationen (IPv4 und IPv6).
