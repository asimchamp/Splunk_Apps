Erklärung der Daten in der Tabelle

Target
Kommunikationsziel (Adresse und Port)
Host
Quellcomputer
Application
Anwendung, die die Kommunikation abwickelte
Process
Prozess, der die Kommunikation abwickelte
User
Benutzerkonto, unter dem der Prozess lief
#Sends
Gesamte Menge der Sendeoperationen
#Receives
Gesamte Menge der Empfangsoperationen
#Connects
Gesamte Menge der Verbindungsoperationen
#Reconnects
Gesamte Menge der Wiederverbindungsoperationen (weil die Verbindung wegen Timeout abbrach oder die Gegenstelle nicht antwortet)
#Retransmits
Gesamte Menge der Erneut-Sende-Operationen (wenn die ursprünglich gesendeten Daten wegen Netzwerkproblemen verloren gingen)
Send volume (MB)
Gesamtes gesendetes Datenvolumen in MB
Receive volume (MB)
Gesamtes empfangenes Datenvolumen in MB
Avg. TCP send latency (ms)
Durchschnittliche Sendelatenz in ms (kann nur für TCP-Verbindungen ermittelt werden)
#Targets
Anzahl unterschiedlicher Ziele
#Hosts
Anzahl unterschiedlicher Hosts
#Processes
Anzahl unterschiedlicher Prozessnamen
#Users
Anzahl unterschiedlicher Benutzer
