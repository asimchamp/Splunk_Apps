Explanation of the data in the table

User
Name of the user logged on to the session.
Host
Name of the computer the session was running on.
Logoff time
Time at which the user logged off
Total duration
Total logoff time in seconds
User profile
Time required for unloading the user profile in seconds.
GP logoff script
Execution time of a logoff script assigned through group policy in seconds. May be empty if no such script exists.
