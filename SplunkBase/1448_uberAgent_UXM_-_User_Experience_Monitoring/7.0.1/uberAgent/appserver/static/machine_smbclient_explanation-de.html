Host

Write IO count

Anzahl Metadata-Operationen (z.B. Verzeichnisauflistungen)
Avg. read latency (ms)

Durchschnittliche Dauer einer Schreiboperation in Millisekunden. Ist leer, falls im gewählten Intervall kein IO anfiel.

Durchschnittliche Anzahl der Leseoperationen pro Sekunde
Avg. write IOPS

Avg. metadata IOPS
Durchschnittliche Anzahl der Metadata-Operationen pro Sekunde
