Explanation of the data in the table

User
User account logged on to the session.
Host
Name of the computer the session was running on.
Logon time
Time at which the user logged on.
Last seen
Last time uberAgent sent data about this session.
Last state
The last recorded session state. Typically either active or disconnected.
Client name(s)
Names of the clients used to connect to the session.
Client IP(s)
IP addresses of the clients used to connect to the session.
Client platform(s)
Platforms of the clients used to connect to the session.
Client version(s)
Version numbers of the Citrix client component (aka Workspace app, Receiver, ..) used to connect to the session.
Pub. name
Name of the published application that was used to start the session.
Pub. app(s)
Names of all published applications running in the session.
Encryption
Encryption level of the remoting protocol.
Policy microphone redirection
Citrix policy to allow/deny microphone redirection.
Policy audio redirection
Citrix policy to allow/deny audio redirection.
Policy audio channel priority
Citrix policy to set the virtual channel priority.
Policy sound quality
Citrix policy to configure the sound quality.
Audio actual channel priority
Applied virtual channel priority.
Policy client drive mapping
Citrix policy to allow/deny client drive mapping
Policy read-only client drive access
Citrix policy to allow/deny read-only client drive access
Policy cdm channel priority
Citrix policy to set the virtual channel priority.
CDM actual channel priority
Applied virtual channel priority.
Mapped volumes
Volumes the client mapped into the session.
Display mode
Active display mode (e.g. Thinwire)
Policy thinwire FPS
Citrix policy to configure the maximum session FPS.
Policy thinwire channel priority
Citrix policy to set the virtual channel priority.
Policy thinwire hardware encoding
Citrix policy to enable/disable hardware encoding.
Policy thinwire video codec
Citrix policy to configure the video codec.
Policy thinwire visual quality
Citrix policy to configure the visual quality.
Thinwire actual channel priority
Applied virtual channel priority.
Thinwire color depth (bit)
Applied session color depth.
Thinwire component encoder
Applied video component encoder.
Thinwire hardware encode
See if hardware encode is actually in use.
Thinwire video codec type
Applied video codec type, e.g. H264
Thinwire color space
Applied session color space, e.g. Yuv420.
Thinwire video codec usage
Applied video codec usage setting, e.g. for actively changing regions.
Policy framehawk priority (deprecated in CVAD 7 1811)
Citrix policy to set the virtual channel priority.
Framehawk actual channel priority (deprecated in CVAD 7 1811)
Applied virtual channel priority.
Policy DCR aero redirection (deprecated in Citrix XA/XD 7.12)
Citrix policy to enable/disable DCR aero redirection.
Policy DCR graphics quality (deprecated in Citrix XenApp and XenDesktop 7.12)
Citrix policy to configure the DCR graphics quality.
Policy DCR channel priority (deprecated in Citrix XenApp and XenDesktop 7.12)
Citrix policy to set the virtual channel priority.
DCR actual channel priority (deprecated in Citrix XenApp and XenDesktop 7.12)
Applied virtual channel priority.
Policy legacy graphics display degrade notify user
Citrix policy to notify the user about display quality degradation.
Policy legacy graphics display degrade policy
Citrix policy to configure the display quality degradation.
Policy legacy graphics channel priority
Citrix policy to set the virtual channel priority.
Legacy graphics actual channel priority
Applied virtual channel priority.
Policy session reliability connections
Citrix policy to enable/disable session reliability.
Policy session reliability port
Citrix policy to configure the session reliability port.
Policy session reliability timeout (s)
Citrix policy to configure the session reliability timeout.
Policy ICA listener port
Citrix policy to configure the ICA listener port.
EDT MTU
Applied Enlightened Data Transport MTU size.
Connected via
IP adress the session connected from.
Transport protocols
The session transport protocol chain.
Policy allow printer redirection
Citrix policy to allow/deny printer redirection.
Policy auto-create client printers
Citrix policy to configure printer auto-creation.
Policy printer channel priority
Citrix policy to set the virtual channel priority.
Printer actual channel priority
Applied virtual channel priority.
Mapped printers
All mapped session printers.
Policy USB PNP redirection
Citrix policy to allow/deny PNP redirection.
Policy USB support
Citrix policy to allow/deny USB support.
Policy USB channel priority
Citrix policy to set the virtual channel priority.
USB actual channel priority
Applied virtual channel priority.
