Explanation of the data in the table

ProgID
ProgID of the plugin (language-independent)
Name
Name of the plugin (may change with version and language)
GUID
GUID of the plugin (language-independent)
Load behavior
Defines how/when the plugin is loaded
All users?
Is the plugin installed for all users?
Load count
Number of times the plugin was loaded
Machine count
Number of machines the plugin was loaded on
Avg. load duration (ms)
Average time in milliseconds it took to load the plugin
Total load duration (s)
Total time in seconds it took to load the plugin
