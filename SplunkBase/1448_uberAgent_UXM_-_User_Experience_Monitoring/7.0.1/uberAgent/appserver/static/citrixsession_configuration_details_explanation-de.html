Erklärung der Daten in der Tabelle

User
Benutzerkonto eingelogged in dieser Sitzung.
Host
Name des Computers, auf dem die Sitzung lief.
Logon-Zeit
Uhrzeit, wann sich der Benutzer angemeldet hat.
Last seen
Zeitpunkt, an dem uberAgent das letzte Mal Daten über diese Sitzung lieferte
Last state
Letzter aufgezeichneter Zustand. Typischerweise active oder disconnected.
Client-Name(n)
Namen des Clients, der für die Verbindung zur Sitzung, verwendet wird.
Client-IP(s)
IP-Addressen des Clients, die für die Verbindung zur Sitzung, verwendet wird.
Client-Plattform(en)
Plattformen des Clients, die für die Verbindung zur Sitzung, verwendet wird.
Client-Version(en)
Versionsnummern der Citrix-Clientkomponenten (aka Workspace app, Receiver, ..), die für die Verbindung zur Sitzung, verwendet wird.
Pub. name
Name der veröffentlichten Anwendung, die für das Starten zur Sitzung, verwendet wird.
Pub. app(s)
Namen von allen veröffentlichten Anwendungen, die in der Sitzung laufen.
Verschlüsselung
Verschlüsselungsstufe des Remoting-Protokolls.
Mikrofonumleitungsrichtlinie
Citrix-Richtlinie zum Einstellen der Mikrofonumleitung.
Audioumleitungsrichtlinie
Citrix-Richtlinie zum Einstellen der Audioumleitung.
