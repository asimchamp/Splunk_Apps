Erklärung der Daten in der Tabelle

#Hosts
Anzahl unterschiedlicher Hosts
#Boot
Anzahl Startvorgänge
Total boot
Gesamte Startzeit bis zu dem Zeitpunkt, an dem der Desktop sichbar ist und das System eine Last von maximal 20% erreicht hat. Dies ist die Summe aus main path und post boot.
Main path
Die Zeit vom Windows-Start bis zum Start der Shell.
Kernel init
Dauer der Kernel-Initialisierung.
Smss init
Smss.exe ist der erste Prozess, der vom System gestartet wird. Enthält die Laufzeit von Autocheck.
Autocheck
Dauer der Festplattenüberprüfung.
Session 0 init
Dauer der Initialisierung der Sitzung 0. Beginnt, wenn smss.exe in Sitzung 0 gestartet wird und endet, wenn wininit.exe gestartet wird.
Session 1 init
Dauer der Initialisierung der Sitzung 1. Beginnt, wenn smss.exe in Sitzung 1 gestartet wird und endet, wenn winlogon.exe gestartet wird.
Wininit init
Dauer der Initialisierung von wininit.exe. Beginnt, wenn wininit.exe in Sitzung 0 gestartet wird und endet, wenn services.exe gestartet wird.
Winlogon init
Dauer der Initialisierung von winlogon.exe. Beginnt, wenn winlogon.exe in Sitzung 1 gestartet wird und endet, wenn logonui.exe gestartet wird.
Autostart services
Dauer des Startens aller Autostart-Dienste.
Computer startup
Gesamte Startzeit bis zu dem Zeitpunkt, an dem die Benutzeranmeldung erfolgen kann.
Post boot
Zeit vom Start der Shell bis zum Absinken der Last auf 20%.
