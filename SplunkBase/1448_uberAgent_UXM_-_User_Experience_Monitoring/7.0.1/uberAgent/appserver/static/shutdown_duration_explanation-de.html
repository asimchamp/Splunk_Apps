Erklärung der Daten in der Tabelle

Time
Die Zeit, zu der der Vorgang des Herunterfahrens begann.
Host
Name des Computers, der heruntergefahren wurde.
Total shutdown
Gesamtdauer des Herunterfahrens. Dies ist die Summe aus User sessions, System session und Kernel.
User sessions
Der für das Beenden der Benutzersitzungen benötigte Zeitraum. In dieser Phase erhalten alle laufenden Anwendungen Benachrichtigungen über das geplante Herunterfahren. Durch nicht reagierende Anwendungen kann diese Phase verlängert werden. Dies Phase enthält User profiles und User policy.
User profiles
Die für das Entladen der Benutzerprofile benötigte Zeit.
User policy
Die für Logoff-Skripte benötigte Zeit.
System session
Die für das Beenden der Sitzung 0 benötigte Zeit. Dies schließt ein Services (pre-shutdown) und Services.
Services (pre-shutdown)
Der Zeitraum, in dem Dienste sich beenden, die sich für Pre-Shutdown-Benachrichtigungen registriert haben.
Services
Die Zeit für das Beenden aller anderen Dienste.
Kernel
Dauer des Beendens des Kernels inklusive aller Treiber.
