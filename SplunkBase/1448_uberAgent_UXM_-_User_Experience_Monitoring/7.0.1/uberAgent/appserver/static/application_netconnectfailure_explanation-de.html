Erklärung der Daten in der Tabelle

App name
Name der Anwendung. Wird von uberAgent automatisch bestimmt.
Process name(s)
Namen der zur Anwendung gehörenden Prozesse, die vergeblich versuchten, Verbindungen herzustellen
Failure count
Anzahl fehlgeschlagener Verbindungsversuche
Target address count
Anzahl unterschiedlicher Zieladressen
Target port count
Anzahl unterschiedlicher Zielports
User count
Anzahl an Benutzern mit Verbindungsfehlern
Source host count
Anzahl an Computern mit Verbindungsfehlern
