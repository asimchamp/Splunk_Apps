Erklärung der Daten in der Tabelle

Process name
Name des abgestürzten Prozesses
Process version(s)
Version des Prozesses, die abstürzten.
App name(s)
Name der Anwendung. Wird von uberAgent automatisch bestimmt.
#Crashes
Anzahl der Crashes für diesen Prozess
User count
Anzahl der betroffenen Benutzer
Host count
Anzahl der betroffenen Hosts
Exception code(s)
Fehlercode
Faulting module(s)
Modul, in dem der Fehler auftrat
Avg. process lifetime
Gesamte Lebensdauer des Prozesses bis zum Fehler
Prozess Identifikator(en)
Windows: Erstellungszeitstempel der Prozess-Binärdatei, extrahiert aus den eingebetteten PE-Header-Informationen
macOS: Eine eindeutige Kennung des ausgeführten Binär-Images
Modul Identifikator(en)
Windows: Erstellungszeitstempel des fehlerhaften Moduls, extrahiert aus den eingebetteten PE-Header-Informationen
macOS: Eine eindeutige Kennung des Binär-Images, in dem der Absturz aufgetreten ist
