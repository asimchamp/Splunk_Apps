Erklärung der Daten in der Tabelle

Avg. CPU (%)
Durchschnittliche CPU-Last auf dem Computer.
Avg. IOPS
Durchschnittliche IOPS-Last auf dem Computer.
Avg. % disk time
Durchschnittliche Auslastung der Festplatte(n) in Prozent.
Avg. IO latency (ms)
Durchschnittliche Latenz aller Lese- und Schreibzugriffe auf die Festplatten.
Avg. RAM (GB)
Durchschnittlich genutzter Hauptspeicher in MB.
Avg. RAM (%)
Durchschnittlich genutzter Hauptspeicher in Prozent des vorhandenen Speichers.
Avg. network (%)
Durchschnittlich genutzte kumulierte Netzwerkbandbreite aller verbundenen Netzwerkadapter.
Avg. Paged pool (MB)
Durchschnittliche Größe des Paged Pool, eines Teils des Kernelspeichers.
Avg. Non-paged pool (MB)
Durchschnittliche Größe des Non-Paged Pool, eines Teils des Kernelspeichers.
Avg. #Handles
Durchschnittliche Anzahl offener Handles (für macOS-Clients: offener Deskriptoren).
Avg. #Threads
Durchschnittliche Anzahl laufender Threads.
Avg. CPU rel. frequency (%)
Durchschnittliche CPU-Frequenz (relativ zur Basisfrequenz).
Last seen
Zeitpunkt, an dem uberAgent das letzte Mal Daten von dieser Maschine lieferte
