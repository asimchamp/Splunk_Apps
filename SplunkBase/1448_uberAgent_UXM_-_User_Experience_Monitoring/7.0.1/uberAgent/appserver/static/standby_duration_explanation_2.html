Explanation of the data in the table

Time
The time the computer started to enter standby.
Host
Name of the computer that was entering / resuming from standby.
Enter standby (s)
The time it took to switch from running state to standby state.
Resume from standby (s)
The time it took to switch from standby state to running state.
Target state
The effective system power state.
Wake source type
The type of source that woke up the machine
Wake source text
Description of the wake up source
Wake timer owner
The application that initiated the wake up
