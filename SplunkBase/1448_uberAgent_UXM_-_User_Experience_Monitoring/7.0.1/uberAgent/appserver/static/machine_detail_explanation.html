Explanation of the data in the table

Avg. CPU (%)
Average CPU load on the computer.
Avg. IOPS
Average IOPS load on the computer.
Avg. % disk time
Average utilization of the disk(s) in percent.
Avg. IO latency (ms)
Average latency of all disk reads and writes.
Avg. RAM (GB)
Average amount of memory used.
Avg. RAM (%)
Average amount of memory used in percent of the installed memory.
Avg. network (%)
Percentage of the cumulated network bandwidth of all connected network adapters used on average during the sample interval.
Avg. Paged pool (MB)
Average size of the paged pool, a kernel memory area.
Avg. Non-paged pool (MB)
Average size of the non-paged pool, a kernel memory area.
Avg. #Handles
Average number of open handles.
Avg. #Threads
Average numer of running threads.
Avg. CPU rel. frequency (%)
Average CPU frequency (relative to the CPU's base frequency).
Last seen
Last time uberAgent sent data from this machine
