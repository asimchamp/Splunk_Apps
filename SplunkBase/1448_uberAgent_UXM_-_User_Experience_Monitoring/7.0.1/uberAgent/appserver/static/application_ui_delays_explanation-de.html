Erklärung der Daten in der Tabelle

App name
Name der Anwendung. Wird von uberAgent automatisch bestimmt.
Process name(s)
Namen der zur Anwendung gehörenden Prozesse, die nicht reagierende Fenster darstellten
App UI unresponsiveness (s)
Zeitdauer in Sekunden, während der die Benutzeroberfläche nicht reagierte
Unresponsiveness count
Wie oft wurde ein Fenster der Anwendung als nicht reagierend erkannt?
Focus count
Wie oft hatte ein nicht reagierendes Fenster den Fokus?
User count
Anzahl an Benutzern, die die Anwendung im nicht reagierenden Zustand verwendeten
Host count
Anzahl an Computern, auf denen die Anwendung im nicht reagierenden Zustand verwendet wurde
