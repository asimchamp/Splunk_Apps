Explanation of the data in the table

App name
Name of the application as determined by uberAgent's algorithms
Process name(s)
Names of those application processes that were displaying unresponsive windows
App UI unresponsiveness (s)
User interface unresponsiveness in seconds
Unresponsiveness count
Number of times a window of the application was unresponsive
Focus count
Number of times an unresponsive window had the focus
User count
Number of users experiencing the application in an unresponsive state
Host count
Number of hosts the application was unresponsive on
