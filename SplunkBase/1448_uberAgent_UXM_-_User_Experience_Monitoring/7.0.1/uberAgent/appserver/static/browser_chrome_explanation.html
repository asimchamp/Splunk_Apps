Explanation of the data in the table

Process type
The type of Chrome process
Total CPU time (s)
Total CPU time.
Avg. CPU (%)
Average CPU load.
Avg. IOPS
Average IOPS.
Total IO (count)
Total number of IO (read and write) requests.
Avg. IO (MB/s)
Average IO volume in MB/s.
Total IO (MB)
Size in MB of all data read from and written to disk.
Avg. IO latency (ms)
Average duration of an IO operation in milliseconds. Is empty if no IO occured during the selected interval.
Avg. RAM (MB)
Average size of the working set (memory usage).
Avg. network (KB/s)
Average transfer rate over the network via TCP and UDP in the selected interval (IPv4 and IPv6).
