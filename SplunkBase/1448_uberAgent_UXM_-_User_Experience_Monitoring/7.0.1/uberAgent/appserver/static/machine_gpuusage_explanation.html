Explanation of the data in the table

Host
Computer name.
GPU name
Name of the display GPU name in this machine.
GPU index
Internal index of the GPU.
GPU compute (%)
Total compute usage of all engines of a GPU.
Dedicated memory (MB)
Dedicated memory usage in MB.
Shared memory (MB)
Shared memory usage in MB.
Dedicated memory (%)
Dedicated memory usage in percent.
Shared memory (%)
Shared memory usage in percent.
