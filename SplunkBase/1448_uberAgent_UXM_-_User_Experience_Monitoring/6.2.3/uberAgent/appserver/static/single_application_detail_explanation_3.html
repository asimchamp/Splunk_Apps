Explanation of the data in the table

Target
Communication target (name/address and port)
Target IP
Communication target IP address
Protocols
Communication protocols used
Send count
Total number of send operations
Receive count
Total number of receive operations
Connect count
Total number of connect operations
Reconnect count
Total number of reconnect operations (happens when the connection is dropped because of timeouts or when the target does not respond)
Retransmit count
Total number of retransmit operations (happens when packets need to be sent again due to network issues)
Send volume (MB)
Total data volume sent in MB
Receive volume (MB)
Total data volume received in MB
Avg. send latency (ms)
Average send latency in ms
Host count
Number of different hosts
User count
Number of different users
