Explanation of the data in the table

Share
Network share the SMB client connected to
Host
Monitored (source) computer
Read IO count
Total number of read operations
Write IO count
Total number of write operations
Metadata IO count
Total number of metadata operations (e.g. directory listings)
Avg. read latency (ms)
Average duration of an read IO operation in milliseconds. Is empty if no IO occured during the selected interval.
Avg. write latency (ms)
Average duration of an write IO operation in milliseconds. Is empty if no IO occured during the selected interval.
Avg. read IOPS
Average number of read IO operations per second
Avg. write IOPS
Average number of write IO operations per second
Avg. metadata IOPS
Average number of metadata operations per second
Read data volume (MB)
Total data volume read in MB
Write data volume (MB)
Total data volume written in MB
Share count
Number of different network shares
Host count
Number of different hosts
