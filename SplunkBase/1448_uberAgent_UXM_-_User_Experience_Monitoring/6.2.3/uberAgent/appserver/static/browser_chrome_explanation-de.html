Erklärung der Daten in der Tabelle

Process type
Chrome-Prozesstyp
Total CPU time (s)
Gesamte verbrauchte CPU-Zeit.
Avg. CPU (%)
Durchschnittlich verbrauchte CPU-Zeit.
Avg. IOPS
Durchschnittlich erzeugte IOPS.
Total IO (count)
Insgesamt erzeugte (Lese- und Schreib-) IOPS.
Avg. IO (MB/s)
Durchschnittlich erzeugtes IO-Volumen in MB/s.
Total IO (MB)
Gesamtmenge des IO-Volumens in MB.
Avg. IO latency (ms)
Durchschnittliche Dauer einer IO-Operation in Millisekunden. Ist leer falls keine IO-Operationen während des gewählten Zeitraums stattfanden.
Avg. RAM (MB)
Durchschnittliche Größe des Working Set (Speicherverbrauchs) der Site.
Avg. network (KB/s)
Durchschnittliche Netzwerk-Übertragungsrate per TCP und UDP im gewählten Zeitraum (IPv4 und IPv6).
