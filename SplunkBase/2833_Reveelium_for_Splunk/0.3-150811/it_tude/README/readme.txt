Introduction:
This Splunk application provides access to Reveelium Smart and Predictive Analytics. It provide APT Threat Model and DNSQuery deviant behaviour analytics by parsing DNS/Proxy Logs without the need to set thresholds.

Note: Without Reveelium API access, your data won't be analysed by Reveelium Smart and Predictive
Analytics solution. Please email support@itrust.fr to request an evaluation key. 

Application Setup:
System requirements: Splunk 6.2.2+

Install the application within Splunk by browsing to Apps > Manage Apps > Find more apps online, and searching for Reveelium, or uploading the package to your Search Head. Once installed, click into the API and Logs Settings tabs within the app.


Troubleshooting and Support: contact@itrust.fr
