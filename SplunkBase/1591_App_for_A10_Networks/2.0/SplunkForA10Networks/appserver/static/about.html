Splunk For A10 Networks provides visibility into the ADC logs, allowing to detect and analyze Web traffic, trends, and Web threats.


Features

  • Server Load Balancing traffic by Request, Response Time, and Size
  • Web Application Firewall stats, such as: Top Allowed/Deniend Clients, Top Policy actions, Top Attacks, and more
  • Field extractions for SLB and WAF logs
