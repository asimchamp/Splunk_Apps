The z/OS System Log (SYSLOG) monitor application allows users to see all messages, or select messages from RACF, CICS, DB2, IMS, USS, WebSphere Application Server and WebSphere MQ. The SYSLOG monitor can also examine some IEF system messages. The z/OS SYSLOG is the main system log that runs on the mainframe. Users can distinguish different message types by examining the different message numbers.

The following are message numbers used in within the application:

Message Number Explanation
ICH408I RACF authority violations
ICH70001I User granted access to system by RACF
IEF450I Job abend
IEF099I Dataset in waiting
IWF403I Job start
IEF404I Job end
IEF453I JCL error
$HASP250 Job was removed from the queue normally
$HASP395 Job completed processing
DSNT375I DB2 deadlock
DSNI031I DB2 lock escalation
DSNB260I Long-running DB2 reader
DSNT376I DB2 time out
