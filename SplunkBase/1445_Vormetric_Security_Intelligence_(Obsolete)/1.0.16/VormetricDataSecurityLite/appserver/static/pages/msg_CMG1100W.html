Message ID

    CMG1100W

Severity

    Warning

Long Form

    {0} certificates will expire {2} {1} days.

Short Form

    cert will expire

    Parameters

      Label CEF Label Type Usage
      cert vds_cert Text Certificate type or name

Description

    Vormetric agents use certificates to authenticate them with the Security Server. These certificates are created during the registration process after the agent is installed and are valid for one year. They automatically renew 60 days before expiration. No action is required.
