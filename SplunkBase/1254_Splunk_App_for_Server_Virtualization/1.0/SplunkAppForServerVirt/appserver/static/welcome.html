Explore Splunk App for Server Virtualization



The Splunk App for Server Virtualization comes with a useful set of views, dashboards and saved searches that provide insights into your virtualization layer. This application should be used in conjunction with the Citrix XenServer Technology Addon and/or the Microsfot Hyper-V Technology Addon. These Technology Addons gather information from your XenServer/Hyper-V hosts, pools, virtual machines, and storage.
