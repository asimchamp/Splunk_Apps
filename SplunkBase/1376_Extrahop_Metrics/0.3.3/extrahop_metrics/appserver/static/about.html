Welcome to the Extrahop companion application: Extrahop Metrics

This application is intended to be a complementary application to the offical Extrahop Splunk application. It is intended to provide a means to collect specific metrics for specific nodes in Extrahop by use of the Extrahop API. Metrics are collected on 5 minute intervals and are the raw metrics prior to any rollup/averaging.

Dependencies:
  • python - most likely an external install as it's hard to get modules installed using the python that comes with splunk
  • pyamf - can be found on the internet
  • pyhop - can be found in this application's main directory
  • dateutils - can be found on the internet
  • sideview-utils - can be installed via the Splunk apps manager

Distributed Deployment Server Conerns
Please not that this application should/need to live on the deployment server so that the lookup tables won't get over written. Management of which metrics should be collected should be done from this server as well.

Setup:
Setup involves visting the setup page either through the manager tab or running this application for the first time. You will need to enter in a few details and once you click save you will be able to configure which nodes and metrics you want to collect.

Searching:
To search for metrics start with the following and filter from there:

index="extrahop_metrics" sourcetype="extrahop_metrics"
