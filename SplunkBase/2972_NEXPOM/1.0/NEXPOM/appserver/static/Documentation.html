<Screen Composition>

This page contains how to use NEXPOM app


1. Data Input
You can input data from measuring equipments.

2. Inspection report
There are report for basic process capability index.

3. Characteristic
This dashboard show some graphs concerning defect rate in process

<Data Input>



It must be done before to see Inspection Report and Characteristic Page.
NEXPOM app provides ‘Cyl_Sample_data’ basically. They will automatically be updated when you install NEXPOM app.


1. Choose process data
Data field must be separated from ‘,’ or ‘blank’.

2. File name & Splunk source type for saving
You can change file name. ‘Blank’ will be convert to ‘_’

3. Imported file content
It needs the value of uppertol, lowertol, nominal and planid to see screen at different page.

4. Save file
If you click this button, the content will be saved at inputs.conf & props.conf.

<Inspection Report>



You can check the process capability index to use Uppertol, Lowertol, Nominal, Actual value of data.
1) Bbasic measuring index : Cp, Cpk, Pp, Ppk
2) Management Chart : X-chart & MR-chart


Note :
1) You must choose Source type & Plan ID
2) In case of Select box, you can get the list by receiving the previous box value
3) The Color of Cp, Cpk, Pp, Ppk will automatically be updated when the interval-valued changes. You might want to seek the ‘single value option’ if you’d like to change the interval-valued.

<Data Report>



This page contains all defect rate, the defect rate per time.
Also you can be aware of how far the measurement result is off the baseline.

Note :
1) You must choose Source type & Plan ID
2) In case of Select box, you can get the list by receiving the value of previous box


These show all defect rate and the defect rate per time of the measurement result.
One is X-chart, and the other is the rate chart out of the reference value.
