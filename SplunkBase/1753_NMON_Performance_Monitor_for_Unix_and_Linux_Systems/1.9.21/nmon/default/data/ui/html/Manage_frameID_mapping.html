FRAMEID MAPPING MANAGEMENT

The frameID mapping management interface allows to configure frameID mapping on a per server basis:

  • logically link a server with a frame identifier
  • to be used as a search filters to filter out groups of server depending in your needs
  • to be used for analytic purposes to perform advanced calculations and aggregations
