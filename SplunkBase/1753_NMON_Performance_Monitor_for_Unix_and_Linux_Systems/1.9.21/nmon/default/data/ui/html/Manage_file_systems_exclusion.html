ALERTING PER SERVER FILE-SYSTEM EXCLUSION

The alerting per server file-system exclusion management interface allows you to configure Nmon alerting features for:

  • exclusion of file-systems from alerting on a per server and per file-system basis
    •

    file-systems exclusion:


  • mount: mount point of the Unix file-system
  • exclude: (true/false) explicit exclusion (default)
    •


    When a server is matched within the nmon_alerting_filesystem_per_server_exclusion KVstore lookup, the file-system will automatically be excluded.

    You can use wildcards to match multiple file-systems at once, and case is insensitive.


    Lookup access:

    | inputlookup nmon_alerting_filesystem_per_server_exclusion
