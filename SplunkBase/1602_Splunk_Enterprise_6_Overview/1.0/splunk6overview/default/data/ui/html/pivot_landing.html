Pivot

Previous: Data Model

Next: Predictive Analytics

Pivot enables Business Analysts and System Admins alike to quickly generate sophisticated visualizations and reports without using the Search Processing Language (SPL). Users can access different chart types from the side bar to visualize their data in many ways.

Queries using the Pivot interface are powered by underlying data models, which are usually designed and implemented by Knowledge Managers familiar with SPL.

Creating a Pivot

After the data model has been created, complete the following steps to create a pivot:
  • Select a data model
  • Select an object to create the pivot
  • Choose a time range filter
  • Select the field(s) to populate the rows - Click the icon in the "Split Row" functionality to select what you want to populate the rows with
  • Select the field(s) to sort the columns by - Use the icon in the "Split Column" functionality to select what you want to populate the rows with
  • Select visualization format
  • Save as a report or dashboard
