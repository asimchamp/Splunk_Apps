New Search Screen Navigation

Splunk 6 has redesigned Search with a cleaner layout that provides improved usability and flexibility. The Search interface unifies searching and reporting and provides easy access to all search features

The New Events Viewer:

  • Extensive time picker provides preset ranges and support for custom ranges
  • Edit search job settings
  • Expanded and robust Events Viewer allows users to drilldown and interact with events and customize their results table while searches continue to run in the background.
  • Quick access to Fields, Field values and Field settings
  • Tabbed results interface

    • Data Summary

    The Data Summary pulls up a window with three tabs and allows you to get summary data on the Hosts, Sources and Sourcetypes of your data.

    Time Picker

    The Time Picker gives you a wide range of ways to select the time ranges for your search. There are presets, Real Time, Date Range, Date and Time Range as well as Advanced options.
