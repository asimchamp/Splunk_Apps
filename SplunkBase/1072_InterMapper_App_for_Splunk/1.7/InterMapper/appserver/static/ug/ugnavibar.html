Using the Navigation Bar

Use the Navigation bar to navigate through the Intermapper Splunk app pages.

  • Maps - Use this dropdown to choose a map imported from your InterMapper server.
  • Notifications - Use this page to view a summary of all notifications generated from device state changes in InterMapper. Click through to view extended information about the device.
  • Device List - Use this page to view a list of all devices being monitored by InterMapper, along with current status of each. IP and MAC address information is included when available. Click through to view extended information about a device.
  • Search - Use this page to search devices and notifications.
  • Default Map - Use this navigation link to return to your specified default map.
  • Devices - Use this dropdown to view links to all of the devices that have custom dashboards within Splunk.
  • Configure App - Use this page to configure your Splunk application.
