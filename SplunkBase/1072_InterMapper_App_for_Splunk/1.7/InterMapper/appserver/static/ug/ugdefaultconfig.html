Configuration Dashboard Overview

Use the Configuration Dashboard to configure the connection to the Intermapper Web Server and to set the default map. Click Configure App in the top menu to view the dashboard.

Intermapper Web Server Connection

  • IP Address - Enter the IP address of the Intermapper web server you want to connect to.
  • Port - Enter a port number to use with the web server connection. Leave the field blank to use the default port for HTTP (80) or HTTPS (443) connections.

App Options

  • Default Map - Enter the name of the Intermapper map you want to use. The value is case-sensitive.

Viewing Switch Information

To include switch information in your Splunk output, add a Splunk/Layer 2 Output probe to a device (one and only one) on a map in Intermapper.
