Alerts Dashboard Overview

Use the Alerts Dashboard to view alerts for a subset of Intermapper notifications received by Splunk.

In the Notifications view or the Device Dashboard, click the number for an entry in the Alerts Summary area. An Alert table shows a list of alerts filtered by the status type you selected.

For example, click "12" in "12 Down Alerts". The Alert table appears, showing you the 12 alerts received when devices went down.

  • Alert Table - View a summary of alerts sent by InterMapper. _time: the time Splunk received the notification the change in device condition. Original Event Time: the time the device condition changed.
  • Time Range - Choose a time range for which to view alerts and events.
  • Alert Events - A list showing a detailed version of each notification event sent by InterMapper.
