Device Dashboard Overview

Use the Device Dashboard to provide view information about a specific device.

  • Device IP - The IP Address of the device
  • MAC Address - The MAC Address of the device
  • Connected to Switch - The IP Address of the switch the device is connected to.
  • Connected to Switchport - The name of the interface the device is connected to.
  • First Detected on Switchport - The time InterMapper first detected the device on this switchport.
  • Last Detected on Switchport - The time InterMapper last detected the device on this switchport. Turns gray when over 7 days old.
  • Device Name - The name of the device as reported by InterMapper
  • Device Alerts - A summary of the InterMapper alerts for the device. Click through for more information.
  • Time Picker - Choose a time range for which to view alerts and events for the device.
  • Events - A list of events within the selected time range that contain either the device name or IP.
