Notification Dashboard Overview

Use the Notification Dashboard to view notifications received from the InterMapper server. View a summary of Alerts and a log of notifications over a selected time range.

  • Time Range - Choose a time range for which to view Notifications.
  • Notification Log - View all notifications received over the selected time range. Use the Send Notification Level filter to limit the data you are viewing to one of these levels: - Down - Up - Critical - Alarm - Warning - OK - Acknowledged
