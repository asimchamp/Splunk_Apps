Splunk for Juniper Firewall

The Splunk for Juniper Firewalls application provides a collection of dashboards and reports to help your organisation monitor for network based attacks as well as help detect malware running on clients systems. The Splunk for Juniper Firewalls application supports Juniper ISG and SSG security devices.


In this release, 1.3.0, we have introduced verification of external IP addresses contacted by systems in your network are not associated with malware or spam bots. This release uses the Project Honey Pot API.


Configuration

The threat score is pulled down from the project honey pot threat intelligence database. To access this database you’ll need to register for free and obtain an authorization code.

  1. Register and optain http:BL API code:
    • https://www.projecthoneypot.org/services_overview.php
  2. Configure http:BL Key
    • Go into $SPLUNK_HOME/etc/apps/SplunkforjuniperFW/bin/
    • Open the scorelookup.py script with an editor
    • Paste your APK Code into the “key” value and save the script
  3. Restart you Splunk instance.
    • From your Splunk web interface click on Manager > Server > Controls > Restart Server
